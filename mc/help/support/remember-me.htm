Remember Me Login Feature

To use the "Remember Me" feature, you will need to allow your Adobe Flash Player to save information on your computer.

  1. Right-click (Windows) or Control-click (Macintosh) the login screen.
  2. From the menu, select "Settings", and then click the file folder (local file storage tab).
  3. Set this to 10kb, then press "close".
