Les jouets t'offrent plein de surprises en ligne !

Club Penguin est un monde virtuel en ligne où les enfants de 6 à 14 ans utilisent leur imagination et créativité pour jouer, explorer et se faire des amis pingouins.

Des surprises en ligne !

Quand tu vois ce symbole , cela veut dire que ton jouet a une pièce en cadeau. Il y a un code au dos de cette pièce qui te permet d'avoir accès à des objets spéciaux en ligne dans Club Penguin.



Utilise ton code jouet.

 

Pour débloquer tes objets, il te faut:

  • Le code que tu as trouvé avec ton jouet.
  • Un compte pingouin. Si tu n'as pas de compte pingouin,
        clique ici     pour créer un compte gratuit.

 

Une fois que tu as ces deux choses, il te faut:

  • Aller à clubpenguin.com/fr et suivre le symbole Débloque des Objets.
  • Suivre les instructions pour entrer ton code et débloquer tes objets.
  • Sélectionner ton serveur et jouer ! N'importe quel objet que tu débloques
        sera automatiquement ajouté à ton compte.


Comment est-ce que je débloque des objets ?

1.

Quand tu vois ce symbole , tu sais que ton jouet a un code qui te permet de débloquer des objets en ligne. Quand tu as un nouveau jouet Club Penguin, garde le code précieusement. Le code ne peut être utilisé qu'une fois alors ne le perds pas !

2.

Clique le bouton Débloque des Objets en haut à droite de ton écran. Si tu n'as pas encore de compte pingouin, crée un compte gratuit en premier.

3.

Connecte-toi avec ton nom de pingouin et mot de passe.

4.

Tu auras besoin de ton code jouet (au dos de la pièce) , ou de ton livre .

5.

Si tu as sélectionné un jouet, entre ton code jouet et clique sur Terminé. Assure-toi de bien entrer le code tel qu'il apparait. (Lire le paragraphe 7)

6.

Si tu as un livre, trouve le mot de code en suivant les instructions dans la question. Assure-toi de bien entrer le code tel qu'il apparaît dans le livre. (Lire le paragraphe 8)

7.

La barre en haut du Livre aux Trésors indique combien d'objets tu peux débloquer. Feuillette le livre et choisis les objets que tu veux.

IMPORTANT : fais bien attention de débloquer tous les objets avec ton code avant de cliquer sur Terminé.

8.

Après avoir répondu correctement à la question secrète, un objet spécial et des pièces seront ajoutés à ton inventaire. Si tu as déjà reçu l'objet spécial, tu auras plus de pièces !

9.

Chaque code doit être entré séparément. Si tu as plus d'un jouet avec un code, ou si tu as un livre, clique sur le bouton Entrer un Nouveau Code.

10.

Une fois que tu as terminé, clique sur Jouer ! Sélectionne ton serveur et va t'amuser !

N'importe quel objet que tu débloques sera automatiquement ajouté à ton compte pingouin.

Que tu aies un abonnement ou un compte gratuit, les objets que tu débloques te seront toujours accessibles !
